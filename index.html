Cara menginstal

  1. Download Apk nya di atas
  2. Pastikan memori penyimpanan kamu tidak penuh
  3. Aktifkan penginstalan sumber tidak dikenal melalui pengaturan keamanan ponsel
  4. Instal Apk nya melalui file manager bawaan atau aplikasi X-plore
  5. Apabila ingin update maka instal langsung saja (ditimpa), jika tidak dapat di-install maka kamu harus menghapus (uninstall) versi sebelumnya terlebih dahulu
  6. Apabila muncul peringatan, lanjut dan klik tetap install
